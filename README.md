## ğŸ“¦ Getting Started

1. **Clone the repo:**

   ```bash
   git clone https://github.com/fredericonicola/portfolio-app.git
   cd portfolio-app
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in your browser:**
   ```
   http://localhost:5173
   ```
